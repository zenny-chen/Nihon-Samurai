个人种田经验分享:

以下仅为个人经验，绝无百分百准确之谈，请各位按需查看。

要种出好的稻子，家里蹲是必不可少的，每1小时就要关注水量、状态等的变化，及时控制水量，及时除草，关注状态。

稻田水量0-25为浅水，50以上为深水，100为湛水。晴天经过6h水量会减少10%左右，雨天则增加10左右。也就是晴天每个小时要放2%的水，雨天每个小时要排2%的水。

农书对各类病害的描述及经SL确认的治疗方式:

- 【徒长】 由于日晒不够导致茎部细长。——『强盛之妙药』+祈求日照天气
- 【抽穗异常】 由于养分不足导致不易结穗。——『强盛之妙药』
- 【倒伏】 由于茎部脆弱或是强风让稻子倾倒，是杂草导致的病害。——『强盛之妙药』+提高拔杂草的频率
- 【盐害】由于稻种含盐量高难以成长。——『剪枝之妙药』
- 【高温伤害】由于土壤或是水温太高难以生长。——『剪枝之妙药』+祈求降雨天气
- 【生长过剩】稻子长得太快造成养分不够。——『剪枝之妙药』
- 【贪青】因稻子未成熟导致米是乳白色的。——稻穗过早插秧导致，待系统提示去插秧时再拿出
- 【胴裂米】精白时碎掉的米——推测为舂米时发生的，未遇到过
- 【稻热病】长微菌所造成的病，会让品质明显下滑。——『守护之妙药』
- 【缟叶枯病】因害虫或杂草引起的病，有时与徒长和抽穗异常有关。——『守护之妙药』
- 【煤烟病】害虫带来的病。会污染叶子和稻穗，使稻子不易生长。——多抓青蛙、蜘蛛等放进田里，每个屋子都要勤检查，不要漏掉男寝与女寝
- 【虱烧】害虫带来的病，会使大量稻子枯萎，品质也会下降。——未遇到过
- 【斑点米】害虫带来的病，米上有类似烧焦的痕迹，会导致品质大幅下滑。——未遇到过
- 【水稻徒长病】长微菌造成稻种生病，插秧后稻子就会枯萎。——『守护之妙药』

## 【win10下解决fatal error的方法】

大名鼎鼎的PVZ，就算没玩过的玩家也应该有所耳闻，不用我多介绍了吧。这里说下如果win10遇到游戏进不去fatal error的话，主要问题就是游戏不能全屏玩。所以WIN+R输入regedit打开注册表，然后找到以下位置：`HKEY_CURRENT_USER\SOFTWARE\SteamPopCap\PlantsVsZombies`，新建DWORD(32位)值，然后改名为`ScreenMode`即可。如果注册表里已有此项的话双击进去，把数值数据改为0，然后再启动游戏就可以进了。

## Windows 11下解决《侍魂晓》（Samurai Rebooot）无法启动的问题

在系统环境变量下添加：**`OPENSSL_ia32cap`**，将其值设置为：**`~0x20000000`**

#### 什么叫科学？（摘自：[德国大学教授如此评价中医，让中国人汗颜](https://links.jianshu.com/go?to=https%3A%2F%2Fwww.toutiao.com%2Fi6723044015240905224)）

在我看来，科学必须符合以下三条标准：
一、以正面经验为基础。“正面经验”是针对确凿的事实而取得的实际效果。正面的事实与主观的臆测是相对立的，离开了事实，科学便失去了形成的必要条件。所以“正面经验”，是经验的事实资料的积累，能够重复和验证的。
二、陈述的单一性。即在一定的上下文意中，具体名词术语的含义是单一的。所陈述的内容都是有一致规定的，并排除其他含义，那怕是稍微相似的含义。
三、经验资料的严格、合理的综合。“严格的”，是指不是任意的、含糊的和近似的；“合理的综合”，是指从收集到的经验资料中建立起合乎逻辑的联系。这种合乎逻辑的联系，就是这个学科的理论体系。这个理论体系，能使人们对未来事物迅速做出有把握的灵活推断，并使原有的结果再度产生。

按照这三条标准，可大致将二十世纪的各种科学分为精密科学、原始科学和伪科学。少数学科属于精密科学，如物理、化学、天文学几乎完全符合三条标准，是精密科学。多数学科只符合第一条，可称为原始科学。离开了取得正面经验的确凿事实，则应称为伪科学。在现代医学（西医）中，绝大多数是原始科学的知识，只有一小部分是精密科学，其中还有较大的伪科学的成份。与此相反，中医除了还有一部份是原始科学和伪科学的残余外，就绝大部分或者主体而言，应当称得上是精密科学。
